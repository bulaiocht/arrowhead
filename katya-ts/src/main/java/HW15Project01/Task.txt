Создать консольное приложение "Телефонная книга"
Приложение должно уметь:
1. Сохранять новый контакт.
2. Искать контакт по имени.
3. Валидировать пользовательский ввод.
4. Экспортировать контакты в простой текстовый формат или в CSV (на выбор).
5. Уметь читать уже сохраненные контакты из простого текстового файла или из CSV при запуске.
6. Использовать конфигурационные файлы (файлы с расширением .properties) для хранения конфигураций.

Для тех кто желает заморочиться:
7. Отправлять email письмо с CSV файлом на указанный почтовый адрес.
Чтобы это сделать необходимо чтобы в вашем build.gradle были добавлены следующие библиотеки в зависимости:
javax.mail и javax.activation
